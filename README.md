## Mappage

공통적인 부분 설명
> 정적인화면
  - 설정버튼, 검색, 내위치이동버튼
> 검색하고 경로찾기

  - 일반적인 네비게이션 사용법과 동일
  - 목적지 설정
  - startnavigation 하면 네비가이드 시작
  - 뒤로가기나 목적지도착30m 부근 가이드 종료

### Emergency User
